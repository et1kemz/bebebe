# Документация приложения Дневник тренировок

## Как правильно использовать приложение.

1. Зайти в приложение и прочтите инструкцию нажав при этом на кнопку (РЕГИСТРАЦИЯ).
2. Если аккаунт уже был создан, то просто введите данные в поля (mail). (Password).
3. Вводите код в приложение на клавиатуре, чтобы его было установлено.
4. Нажмите кнопку «Вход» для входа в систему.
5. После ввода пароля введите ваши данные.
6. Нажмите кнопку «Зарегистрироваться».
7. Ввести email-адрес и пароль для регистрации.
8. Нажмите кнопку «Зарегистрироваться».

## Проблемы при регистрации или входе

1. Почта уже занята:
   * Сообщение системе: «Пользователь с таким email уже существует».
   * Решение: Предложить пользователю нажать кнопку «Вход» или восстановить пароль.
2. Пароль слишком простой:
   * Сообщение системе: «Пароль должен содержать минимум 6 символов».
   * Решение: Попросить придумать более надежный пароль и показать требования (например, цифры, буквы).
3. Неверный формат почты:
   * Сообщение системе: «Введите корректный email-адрес (например, name@example.com)».
   * Решение: Подсветить поле Email красным и показать пример.
4. Пользователь забыл нажать «Регистрация» после ввода кода:
   * Напоминание: В инструкции (пункт 6 и 8) кнопка дублируется, чтобы точно не забыли.



## Как правильно использовать приложение для (Администратор).

1. Администратор может так же может использовать приложения как и обычный пользователь.
2. Администратор может блокировать аккаунты.
3. Администратор может ставить упражнения в стоп-лист, стоп-лист это временное исключение упражнения из списка рекомендованных (например, при травме или поломке тренажера).
4. Так же администратор имеет доступ к базе данных приложений._

## Описание приложения

Тип приложения: база данных / дневник / справочник упражнений.
Целевая аудитория: новички в зале, спортсмены, люди, восстанавливающиеся после травм, тренеры.
Платформы: Android / iOS / Web (по плану).
Интерфейс: минималистичный, с быстрым поиском упражнений и карточками с техникой выполнения.

## Преимущества

Вся ваша тренировочная программа — в одном месте. Быстро находишь упражнение на нужную группу мышц благодаря поиску и фильтрам. Удобно следить за прогрессом (веса, повторы) и не забывать, что делал на прошлой тренировке. Можно вести личный дневник, а не зависеть от готовых шаблонов из интернета.
